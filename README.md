# coprogen

An Android ContentProvider generator.

## Getting Started
Install with npm:
```bash
$ npm install
```
Link the command line tool (coprogen):
```bash
$ npm link
```
Run `coprogen` in the directory that contains your content-provider.json file:
```bash
$ coprogen
```
